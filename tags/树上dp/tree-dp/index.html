Codeforces 1187E - Tree Painting

换根有点意思的。 Solution We can calculate the point if we fix the starting node in $O(n)$ time. In order to get the final answer, we need to calculate all the points of all the starting nodes, which will give us a $O(n^2)$ solution, too slow. Actually, we only need to calculate the point once and we can easily get the points for other staring points: Suppose we have calculated the $point$ of some start vertex $u$ and we want to know the point for some other starting vertex $v$.
