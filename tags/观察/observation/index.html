Solution of Codeforces 1238D - AB-string

还是得多观察找规律。 Solution Instead of counting good substrings, we could count bad substrings. Let’s call a character $t_i$ in string $t_1\dots t_k$is bad if it doesn’t belong to any palindrome. The key observation is that all the characters in $t_2\dots t_k-1$ are good, here’s the proof: There are only two cases: $t_i=t_{i+1}$ or $t_i=t_{i-1}$, in this case $t_i$ belongs to a palindrome of length 2. $t_i\ne t_{i-1}\ne t_{i+1}$.
