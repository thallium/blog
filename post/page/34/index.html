Let $dp_{i, j}$ be the number of ways to draw circles whose right boundary is $i$, with mask of $j$. 贪就完事了 Solution First let’s find $s_i$: the number of friends who love food $i$. For some food $i$, if $s_i\leq w_i$, we can see that all the friends who love $i$ will have food to eat no matter what order you call them. So we want to call them as late as possible. The solution is like doing a topological sort or BFS: we start from all the $i$ that $s_i\leq w_i$.
