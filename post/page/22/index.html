Of course there is shorter solution but is more difficult to understand. First lets define the dp status, let $dp_{i,j}$ be the answer for the fist $i$ elements whose index of the last chosen number is $i-j$. The key observation is that if $i$ is odd, $j\leq 2$, if $i$ is even $j\leq 1$.
